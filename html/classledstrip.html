Detailed Description

ledstrip ADT for animations

This ADT lets you control a ledstrip with very easy functions.
This ADT is focussed on animations.
If you want to use advanced functions please refer to ledstrip_array or ledstrip_port.
NOTE: There is NO limitation of ledcount.

Fill the first couple leds with a given color.

Example: fillleds(hwlib::color(255,0,0), 40)
The first 40 leds will turn red, the following leds will be turned off.



More colors? It is also possible to place colors back to back:
fillleds(hwlib::color(255,0,0), 40, 0) wont turn off the leds after the 40 red leds.
Directly after needs to be another fillleds() to turn the rest of the leds another color.
The time between the funtions needs to be less then 50 us.

Flow animation.

Create a flow animation by shifting every led to the next led.
With ledson and ledsoff the user can decide how many leds will be turned on and off after each other.
The fastness is the time between 1 shift. int flows is the amount of flows.

Rainbow animation.

This functions creates a rainbow effect by cycling through all colors.
An int fastness of 5000 is recommended. Lowering it by a couple hundred will result in a faster cycle.
Int rainbows is the total of cycles.
