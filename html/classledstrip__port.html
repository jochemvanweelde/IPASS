Detailed Description

ledstrip_port for using multiple ledstrips as one.

Add multiple ledstrip_pin min 1 and max 16 ledstrip_pin.
This class works with ledstrip_array

change colors of leds

This function makes it possible to change colors in a port
beginled is the first led that is going to change
endled is the last led that is going to change.
NOTE: The ledcount continues on the next ledstrip.
Example (20 led ledstrip, 10 led ledstrip):
changegroupled(15,25,green) will change the last 5 leds of the first ledstrip green
AND the first 5 leds of the second ledstrip green.
This array will also be updated in the pin itself.

Implements ledstrip_pin.

Write a 1 or a 0 to the whole ledstrip_port.

write(1) will result in putting every color stored in the structws2812 array on every ledstrip. (Variable on)
write(0) will result in putting every color stored in the structws2812 array on every ledstrip. (Variable off)
To change the colors go to changegroupled()
