Detailed Description

ws2812led

This class stores 1 ws2812led and lets the user control it with multiple functions.
Example: ws2812led name(due::pin_out(due::pins::##), hwlib::color(###,###,###));

Change the color of the led.

changecolor(hwlib::color(###,###,###)) changes the color of the led to the color that is given.
It's also possible to change the 'off-color' with changecolor(hwlib::color(), 0).
If you write a 0 to the led, it will change to that color.

Write a 1(on) or 0(off) to the led.

write(1) will result in the led turning to the color given in the construcor or with the function changecolor().
write(0) will result in the led turning off or the given color in the function changecolor().
