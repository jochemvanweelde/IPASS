Detailed Description

ledstrip with array (w2812)

This ADT lets the user store a ledstrip and control it.
This ADT can store a max of 120 leds on a ledstrip.
NOTE: The max can be changed by changing the value of the struct length.

Snake animation.

This function adds 1 led at a time
For example: snake(50, 2) will result in:
2 snakes and every led will be added each 50 miliseconds.
Animation: led 1, led 1 and 2, led 1 and 2 and 3 etc.
The leds wont be turned off at the beginning of this animation.

Reimplemented in ledstrip_invert, and ledstrip_dummy.

◆ write()

virtual void ledstrip_array::write ( bool  x)
inlinevirtual

Write a 1 or a 0 to the whole ledstrip.

write(1) will result in putting every color stored in the structw2812 array on the ledstrip. (Variable on)
write(0) will result in putting every color stored in the structw2812 array on the ledstrip. (Variable off)
To change the colors go to changeled() for 1 led or changegroupled() for more.

Reimplemented in ledstrip_invert, and ledstrip_dummy.
