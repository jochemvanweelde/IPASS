Detailed Description

structw2812 (struct)

This struct is used for ADT ledstrip_array.
It stores information of 1 led: The 'on color' and the 'off-color'.
Create a color combination with: structw2812 NAME(hwlib::color(), hwlib::color());
The standard value is white(on) and black(off).
For more information, go to ledstrip_array -> changeled() or changegroupled()
